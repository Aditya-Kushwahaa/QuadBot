# Obstacle Avoidance Mode

QuadBot navigates autonomously using an HC-SR04 ultrasonic sensor.

## Features

- Forward Movement
- Obstacle Detection
- Automatic Reverse
- Random Direction Selection
- Autonomous Navigation

## Required Libraries

- Servo
- FlexiTimer2
- NewPing

## Sensor Pins

TRIGGER_PIN = 19

ECHO_PIN = 18

## Usage

Upload the code and place the robot on a flat surface.

QuadBot will continuously move while avoiding obstacles.
