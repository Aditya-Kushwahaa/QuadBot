# Bluetooth Control Mode

Control QuadBot wirelessly using an Android smartphone and HC-05 Bluetooth module.

## Features

- Forward
- Backward
- Left
- Right
- Hand Wave
- Hand Shake
- Dance

## Commands

| Command | Action |
|----------|---------|
| F | Forward |
| B | Backward |
| L | Left |
| R | Right |
| W | Wave |
| U | Handshake |
| V | Dance |

## Required Libraries

- Servo
- FlexiTimer2

## Usage

1. Pair phone with HC-05.
2. Open control app.
3. Connect via Bluetooth.
4. Send commands.
