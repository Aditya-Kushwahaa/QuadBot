# Basic Movement Mode

This mode demonstrates the basic locomotion capabilities of QuadBot.

## Features

- Stand Up
- Walk Forward
- Walk Backward
- Turn Left
- Turn Right
- Hand Wave
- Hand Shake
- Body Dance
- Sit Down

## Required Libraries

- Servo
- FlexiTimer2

## Usage

1. Open Basic_Movement.ino
2. Select Arduino Nano
3. Upload code
4. Power the robot

The robot will continuously perform a sequence of predefined movements.
